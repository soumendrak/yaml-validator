yaml-validator

Validate YAML files for syntax, style, and schema correctness

GitHub MIT License Python 3.10+ PRs Welcome

Features

Syntax Validation Catches malformed YAML with precise line & column reporting
Style Checks Trailing whitespace, tab indentation, line length, EOF newlines
Schema Validation Validate structure against JSON Schema or YAML schema files
CI Ready JSON output mode, reliable exit codes, quiet mode

Quick Install

pip install git+https://github.com/soumendrak/yaml-validator.git

or with uvx (no install):

uvx --from git+https://github.com/soumendrak/yaml-validator.git yaml-validator check file.yaml

Usage

$ yaml-validator check config.yaml
✓ config.yaml

$ yaml-validator check bad.yaml
[ERROR] line 3:1: found character '\t' that cannot start any token

✗ bad.yaml
   1 error(s)

$ yaml-validator check data.yaml --schema schema.json --json
{"valid": false, "errors": [...]}

Exit Codes

 0  All files pass validation
 1  One or more files failed
